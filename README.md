# MOX Magazine Demo

Demo estatica de MOX como revista vertical de proyectos arquitectonicos.

## Como abrir

Abre `index.html` en el navegador.

## Incluye

- Tres proyectos editoriales en formato slide vertical.
- Tarjetas con datos clave sobre cada proyecto.
- Boton `Ver mas` con imagen principal, render de potencial, concepto de desarrollo y ficha resumida.
- Boton `Reservar` conectado a WhatsApp: +57 321 993 1029.
- Imagen principal basada en la referencia entregada.

## Estructura por proyecto

La carpeta madre real esta en `C:\Users\batos\OneDrive\Desktop\FELO\MOX`.

Cada proyecto debe tener una carpeta numerada:

```text
001/
  principal.png
  potencial.png
  desarrollo.png
  ficha.txt
```

## Publicacion gratuita

Esta carpeta se puede subir tal cual a GitHub Pages, Netlify, Vercel o Cloudflare Pages.
